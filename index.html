PDC Assignments

Welcome to PDC Assignments, a curated collection of computer programming assignments related to parallel and/or distributed computing (PDC). These assignments provide hands-on experiences with different aspects of PDC, including:
  • Distributed-memory parallel computing via the Message Passing Interface (MPI).
  • Shared-memory CPU parallel computing via OpenMP, C++11, and OpenACC multithreading.
  • GPU parallel computing via CUDA, OpenACC, and OpenMP.
All of the exercises can be completed using C or C++; the distributed-memory exercises may be completed using Java (via OpenMPI's Java bindings) or Python (via mpi4py).
