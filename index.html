PDC Assignments

Welcome to PDC Assignments, a curated collection of computer programming assignments related to parallel and/or distributed computing (PDC). The goal is to provide hands-on experiences in which students apply particular PDC concepts and abstractions.

The Assignments

The collection currently contains the following assignments; each assignment provides a sequential solution that the students must speed up using PDC techniques.

Assignment Problem Overview Domain
Boids Compute and visualize the flocking behavior of virtual birds called bird-oids, or boids for short. AI (Rules-based)
Circuit Solver Given a 32-bit digital circuit, find its solutions--all the inputs that cause the circuit to produce the value 1 (as opposed to 0). Computer Engineering
Forest Fire Using a Monte Carlo simulation, find the relationships between the dryness of a forest and (i) the percentage of the forest that burns, and (ii) how long the fire burns before it burns out. Environmental Management
GalaxSee. Using N-body problem techniques, compute and visualize a galaxy of N stars to see how the stars' positions change over time. Computational Physics
Genome Scanner Find how many times a genetic sequence occurs within an organism's genome, and the locations of those occurrences within the genome. Computational Biology
Image Processing Compute and visualize image-conversion operations such as inversion, gray-scale, blur, sharpen, and so on. Photography
The Mandelbrot Set Compute and visualize the figure discovered by Benoit Mandelbrot. Nonlinear Mathematics
Matrix Algebra Perform matrix operations such as addition, transpose, and multiplication. AI / Mathematics
PIN Finder An employee has forgotten their personal identification number (PIN), but their encrypted PIN is available. Decrypt their encrypted PIN. Computer Security
Sum of the Squares The sum of the squares measures the variance of a dataset's numbers around its mean. Compute the sum of the squares of a dataset's numbers, which are stored in an input file. Data Science / Statistics

If you have an assignment you have found to be especially effective for helping students apply PDC concepts, please contact us.

Software/Hardware Platforms

Since different instructors have different technology preferences and/or teaching environments, the following links organize the assignments by their software/hardware platforms:

  • MPI (the Message Passing Interface) assignments can be used to explore the parallel capabilities of diverse multiprocessors, including single-board computers, laptops, desktops, networks of workstations (NoWs), Beowulf clusters, and supercomputers.
  • CPU (multithreading) assignments can be used to explore the parallel capabilities of a computer's multicore central processing unit (CPU), including single-board computers, laptops, desktops, and workstations. Supported multithreading platforms include OpenMP, C++11, and OpenACC.
  • GPU assignments can be used to explore the parallel capabilities of a computer's graphics processing unit (GPU). Supported software platforms include CUDA, OpenACC, and OpenMP.
All of the assignments can be completed using C or C++; the distributed-memory exercises may be completed using Java (via OpenMPI's Java bindings) or Python (via mpi4py).
