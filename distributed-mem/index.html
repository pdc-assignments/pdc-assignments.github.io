PDC Assignments: Distributed-Memory Parallel Computing

The assignments listed below provide hands-on experiences for parallel computing in distributed-memory systems, such as supercomputers, Beowulf clusters, or a network of workstations (NoW), using the Message Passing Interface (MPI). In each assignment, a sequential solution is provided; the challenge is to use parallelism to speed up that solution.

  1. Circuit Solver.
    Using a brute-force approach, find all of the inputs that cause a 32-bit digital circuit to produce the value '1'.
  2. The Mandelbrot Set.
    Plot the set discovered by Benoit Mandelbrot.
  3. Forest Fire Simulation.
    Using a Monte Carlo simulation of a forest fire, find the relationships between the dryness of a forest and (i) the percentage of the forest that burns, and (ii) the length of time the fire burns.
  4. The Pin Finder.
    Using a brute-force approach, decrypt an employee's encrypted personal identification number (PIN).
  5. Sum of the Squares.
    Using parallel techniques, speed up the computation of the sum of the squares of the values stored in an input file.
  6. Genome Scanner.
    Find the locations of a specific genetic sequence within an organism's genome.
  7. GalaxSee.
    Determine what happens to a galaxy's stars over time using an N-body galactic simulation.
Note that MPI solutions also run efficiently on multicore desktops and/or laptops--but they solve a problem using multiple processes that communicate through message-passing (instead of multiple threads that communicate through shared memory).
